# E-commerce Sales Analysis

## Objective
Analyze customer and city-wise revenue to identify key business insights.

## Tools Used
- Python (Pandas)
- SQL
- Power BI

## Key Insights
- Mumbai is the top revenue-generating city
- Revenue is concentrated among a few high-value customers
- Ankit is the only VIP customer

## Features
- Customer segmentation (VIP, Regular, Low)
- Revenue contribution analysis
- Top customer and city identification

## Files
- analysis.py → Python analysis
- queries.sql → SQL queries
- dashboard.pbix → Power BI dashboard
- sales_data.csv → dataset
